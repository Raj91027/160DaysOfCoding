# 160 Days of Coding

Welcome to the **160 Days of Coding** repository! This repository is a collection of coding problems and solutions organized by topics, aimed at improving problem-solving skills and preparing for technical interviews.

## Folder Structure

The repository is organized into topic-specific folders. Each folder contains solutions to problems related to that topic. Here's an overview of the structure:
160DaysofCoding/ │ ├── .dist/ # Build or compiled files (if applicable) ├── .vscode/ # VS Code workspace settings │ └── settings.json ├── Array/ # Problems related to arrays │ ├── Day1-SecondLargestNumber.cpp │ ├── Day2-MoveAllZeroesToEnd.cpp │ ├── Day3-ReverseArray.cpp │ ├── Day4-RotateArray.cpp │ ├── Day5-NextPermutation.cpp │ ├── Day6-MajorityElement-II.cpp │ ├── Day7-StockBuyandSell.cpp │ ├── Day10-MaximumSubArraySum.cpp │ ├── Day11-MaximumSubarrayProduct.cpp │ ├── Day12-MaximumCircularSubArraySum.cpp │ ├── Day13-SmallestMissingPositiveNumber.cpp │ └── ... ├── Hashing/ # Problems related to hashing ├── LinkedList/ # Problems related to linked lists ├── Matrix/ # Problems related to matrices ├── PrefixSum/ # Problems related to prefix sums ├── RecursionAndBackTracking/ # Problems related to recursion and backtracking ├── Searching/ # Problems related to searching algorithms ├── Sorting/ # Problems related to sorting algorithms ├── Stack/ # Problems related to stacks ├── String/ # Problems related to strings └── TwoPointerTechnique/ # Problems related to the two-pointer technique



## How to Use

1. Navigate to the folder corresponding to the topic you want to explore.
2. Open the `.cpp` file for the specific day to view the problem solution.
3. Compile and run the code using your preferred C++ compiler.

## Topics Covered

- **Array**: Problems involving arrays, such as finding the second largest number, reversing arrays, and solving subarray problems.
- **Hashing**: Problems involving hash maps and hash sets.
- **Linked List**: Problems involving singly and doubly linked lists.
- **Matrix**: Problems involving 2D arrays and matrix operations.
- **Prefix Sum**: Problems involving prefix sum techniques.
- **Recursion and Backtracking**: Problems involving recursive algorithms and backtracking.
- **Searching**: Problems involving binary search and other searching techniques.
- **Sorting**: Problems involving sorting algorithms.
- **Stack**: Problems involving stack data structures.
- **String**: Problems involving string manipulation.
- **Two Pointer Technique**: Problems involving the two-pointer approach.
